# Calculator Web App

## Overview
        A simple calculator web application developed using HTML, CSS, and JavaScript. The application performs basic arithmetic operations through an interactive and user-friendly interface.

## Features
>> Addition
>> subtraction
>> Multiplication
>> Division
>> Clear Function
>> Responsive USer Interface

## Technologies Used
* HTML
* CSS
* JavaScript

## Files Included
* index.html
* style.css
* util.css
* script.js

## Project Purpose
        This project was developed to strenghthen front-end web development skills and improve understanding 
of JavaScript logic and DOM manipulation.
